CSS (Cascading Style Sheet)

There are three ways to use css in html. Those are

1. Inline CSS

Code

<h4 style="color:red">This is Inline css</h4>

Result

This is Inline css

1. Internal CSS

Code

<style> .internal{color:green;} </style> <body> <h4 class="internal">This is Internal css</h4> </body>

Result

This is Internal css

1. External CSS

Code

<link rel="stylesheet" type="text/css" href="css/style.css"> <body> <h4 class="external">This is external css</h4> </body>

Result

This is external css
